# english_study_book
英语自检应用

## 接口

### 登录系统

| 接口           | 方法 | 传参     | 接口描述   |
| -------------- | ---- | -------- | ---------- |
| /user/login    | POST | 登录表单 | 登录账号   |
| /user/register | POST | 注册表单 | 注册账号   |
| /captcha/get   | GET  | 无       | 获取验证码 |
| /captcha/check | GET  | 验证码   | 校验验证码 |
